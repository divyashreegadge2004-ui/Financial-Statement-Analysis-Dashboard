# 📊 Financial Statement Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard provides a comprehensive analysis of a company's financial performance.

## 📈 Dashboard Features
- Revenue Analysis
- Gross Profit
- Net Profit
- Gross Margin %
- Net Profit Margin %
- KPI Cards
- Monthly Revenue Trend
- Expense Analysis
- Financial Performance Overview
- Interactive Filters and Slicers

## 🛠 Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## 💡 Skills Demonstrated
- Data Cleaning
- Data Modeling
- DAX Calculations
- Financial Reporting
- KPI Design
- Dashboard Development
